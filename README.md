# leboiko/homebrew-tap

Homebrew formulae for my [OSS projects](https://github.com/leboiko).

## Available

- **[agentop](https://github.com/leboiko/claude-codex-pid-inspector)** — a TUI
  process inspector for Claude Code and OpenAI Codex CLI.

## Install

```sh
brew install leboiko/tap/agentop
```

Or tap first, then install:

```sh
brew tap leboiko/tap
brew install agentop
```

## Updating

Formulae in this tap are regenerated automatically by each project's release
workflow. See the relevant project for how to trigger a new version.
